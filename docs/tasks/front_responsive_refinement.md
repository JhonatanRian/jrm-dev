# Task: Refinamento de Responsividade Neo-Brutalista

## Descrição
Ajustar o layout para garantir que as bordas grossas e sombras sólidas não comprometam a usabilidade em telas pequenas.

## Sub-tarefas
- [ ] Revisar o espaçamento (padding/margin) em dispositivos móveis no `portfolio.css`.
- [ ] Ajustar o tamanho das sombras (`neo-shadow`) para telas menores (shadows muito grandes podem poluir o layout).
- [ ] Implementar menu hamburguer personalizado com estilo brutalista para o `header.html`.
- [ ] Testar a legibilidade dos cards de projetos em resoluções de 320px a 480px.

## Critérios de Aceite
- Layout sem overflow horizontal em telas pequenas.
- Elementos clicáveis com tamanho adequado para toque.
- Menu mobile funcional e estilizado conforme o tema.
