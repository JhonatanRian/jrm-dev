# Task: Implementação de Dark Mode (Neo-Brutalismo)

## Descrição
Implementar a funcionalidade de alternância de tema (claro/escuro) adaptando a estética Neo-Brutalista para usar cores neon e contrastes profundos.

## Sub-tarefas
- [ ] Definir a paleta de cores Dark Neon (ex: sombras pretas puras, fundos cinza escuro, destaques em ciano ou verde limão).
- [ ] Adicionar lógica de JavaScript para persistir a preferência do usuário no `localStorage`.
- [ ] Criar variáveis CSS (Custom Properties) para cores de fundo, bordas e sombras.
- [ ] Implementar o componente de Toggle (botão de troca) no `header.html`.
- [ ] Ajustar as classes do Tailwind para suportar o prefixo `dark:`.

## Critérios de Aceite
- Troca de tema sem recarregamento de página.
- Preferência do usuário salva para visitas futuras.
- Estética Neo-Brutalista mantida e legível em ambos os modos.
