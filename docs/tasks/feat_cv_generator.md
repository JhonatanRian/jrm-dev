# Task: Gerador de Currículo Dinâmico (PDF)

## Descrição
Criar uma funcionalidade que gere um arquivo PDF do currículo automaticamente, utilizando os dados já cadastrados no banco de dados.

## Sub-tarefas
- [ ] Escolher e configurar biblioteca de PDF (`WeasyPrint` ou `ReportLab`).
- [ ] Criar um template HTML específico para a estrutura do currículo.
- [ ] Implementar uma View que receba os dados do `Portfolio` ativo e renderize o PDF.
- [ ] Adicionar um botão "Download CV" no cabeçalho ou hero do portfólio.

## Critérios de Aceite
- PDF gerado com layout limpo e profissional.
- Dados sincronizados com o conteúdo atual do portfólio.
