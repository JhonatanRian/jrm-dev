# Task: Estilização do Feed de Integrações (Jira/Gemini)

## Descrição
Criar uma interface visual no painel (ou seção privada) para exibir o status das integrações configuradas no app `core`.

## Sub-tarefas
- [ ] Criar componentes de "Card de Status" para cada integração (Jira, Gemini, etc.).
- [ ] Implementar indicadores visuais de "Conectado" ou "Erro de Autenticação".
- [ ] Adicionar um estilo de "terminal" para logs de sincronização, mantendo a coerência com o Neo-Brutalismo.
- [ ] Criar formulários de configuração que utilizem o `crispy_neurobrutalist`.

## Critérios de Aceite
- Visual claro sobre quais integrações estão ativas.
- Feedback visual imediato ao salvar novas credenciais.
